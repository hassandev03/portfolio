    RecipeRealm

    RecipeRealm is a feature-rich Flutter recipe app powered by Firebase for seamless backend integration, real-time streams, and data persistence. Built using Riverpod for state management, the app includes a carousel view for categories, a favorites section, and multiple meal categories to explore. Users can sign up or log in with email/password or Google authentication . The app allows users to add, edit, and manage their own recipes, while they can only view recipes added by others. The smooth UI and real-time updates make recipe discovery and management effortless!.

    EzzyShop

    EzzyShop is a sleek and modern Flutter app powered by Provider for smooth state management and real-time API integration using HTTP methods. It features a dynamic product listing, a favorites wishlist , and an intuitive shopping cart with a badge counter , ensuring a seamless shopping experience. The responsive grid layout, elegant UI, and bottom navigation bar make browsing effortless. With smooth animations and instant UI updates, EzzyShop delivers a premium, user-friendly shopping experience.

    ExpenseMate

    ExpenseMate is a powerful Flutter expense tracker that offers a seamless budgeting experience with both light/dark mode UI, a side drawer for easy navigation, and interactive charts to visualize spending patterns. Users can efficiently manage their expenses with a modal bottom sheet for quick entry, a favorites feature for frequently tracked expenses, and categorized spending with intuitive icons. The app's bottom navigation bar provides smooth access to Home, Favorites, and Settings, making financial management effortless and visually engaging.

    Quiz App

    CUBot

    CUBot is an innovative desktop chat assistant built with Java Swing and AWT that effortlessly blends practical functionality with engaging AI-driven conversation. It features a dynamic chat interface where users can launch applications, open text files, capture screenshots, and even check the weather or calculate BMI through simple commands, all while receiving witty, context-aware responses. By integrating APIs like OpenWeatherMap and OpenAI Chat, CUBot not only streamlines everyday tasks but also showcases a creative fusion of automation and intelligent interaction.

    TextWise

    TextWise is a sophisticated text management system built in C++ that seamlessly integrates robust editing capabilities with modern compression and encryption techniques. It allows users to open, edit, and analyze text files—featuring word and character counts, search and replace functions, and intuitive undo/redo support via a custom stack—while also leveraging a Huffman coding algorithm and min-heap structure for efficient file compression and encryption. This creative blend of data structures and user-centric design transforms routine text processing into a secure and intelligent document management experience.
